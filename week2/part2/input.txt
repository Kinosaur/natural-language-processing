Invalid URLs (missing scheme, wrong scheme, incomplete):
- www.example.com (missing scheme)
- example.com (missing scheme)
- mailto:user@example.com (wrong scheme - not https/http/ftp)
- file:///path/to/file (wrong scheme)
- https:// (incomplete - no hostname)
- htp://example.com (typo in scheme)

Invalid Email Addresses (wrong TLD, missing parts, invalid format):
- user@example.xyz (wrong TLD - only .com/.org/.edu/.net allowed)
- admin@site.co.uk (wrong TLD)
- invalid@domain.info (wrong TLD)
- no-at-sign.com (missing @ symbol)
- @nodomain.com (missing username)
- user@ (missing domain)
- user@@example.com (double @)
- user @example.com (space in username)

Invalid Hashtags (with special chars, punctuation, spaces):
- # NoHashtag (space after #)
- #tag! (exclamation should not be included)
- #test@mention (@ should terminate hashtag)
- #tag. (period should terminate)
- #price$99 ($ should terminate)
- #discount%off (% should terminate)
- #two&more (& should terminate)
- #question? (? should terminate)
- #end, (comma should terminate)
